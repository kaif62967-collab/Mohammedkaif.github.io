# 💳 Credit Card Financial Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze credit card transactions, customer behavior, and financial performance. The dashboard provides business insights through KPIs, charts, and interactive filters.

---

## 🎯 Objectives

- Analyze credit card transaction data
- Monitor revenue and customer trends
- Compare card categories and customer segments
- Create an interactive business dashboard

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📊 Dashboard Features

- Revenue Analysis
- Transaction Analysis
- Customer Segmentation
- Card Category Analysis
- Weekly Performance
- Interactive Filters & Slicers

---

## 📈 Key KPIs

- Total Revenue
- Total Transactions
- Total Customers
- Average Revenue
- Revenue by Card Type
- Revenue by Customer Segment
